# CraftStudio
CraftStudio est une app macOS épurée pour vos projets créatifs. Vectorisez vos images en SVG ou PNG, nettoyez vos photos avec l'outil gomme et convertissez vos MP4 en MP3 en un clic. Une interface style Cricut, centrée sur le glisser-déposer, pour un flux de travail fluide, élégant et sans superflu.
